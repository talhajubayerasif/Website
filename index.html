Projects

Personal website

This website serves as a dynamic and comprehensive showcase of my skills, projects, and personal journey in the world of technology. Developed from the ground up, this project was an exercise in building a modern, responsive, and aesthetically pleasing single-page application.

Key Features & Technologies:

  • HTML & CSS: The foundational structure and design of the website were meticulously crafted using HTML and CSS. I focused on creating a clean, professional layout with smooth transitions, responsive design elements, and a cohesive color palette.
  • JavaScript: The user experience was enhanced with JavaScript, specifically for implementing a smooth-scrolling effect between sections. This feature allows for seamless navigation, making it easy for visitors to explore different parts of the portfolio.
  • Planned PHP Integration: Future plans for this project include the addition of a backend with PHP. This will enable functionality such as a working contact form that sends messages directly to my email, further improving the site's interactivity.

This project not only demonstrates my proficiency in front-end development but also my ability to plan, structure, and enhance a web application with a focus on both design and functionality. It is a living document of my skills and serves as a launching point for future improvements and features.
